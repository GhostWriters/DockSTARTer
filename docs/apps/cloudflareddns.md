## Description

[CloudFlareDDNS](https://www.cloudflare.com/dns/) is a small Docker image that
allows you to use the free CloudFlare DNS Service as a Dynamic DNS Provider
([DDNS](https://en.wikipedia.org/wiki/Dynamic_DNS)).
